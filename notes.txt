# train the models

## install required software
- docker engine
- nvidia-container-runtime (may depend on the system)

on arch follow the guide https://wiki.archlinux.org/title/Docker#With_NVIDIA_Container_Toolkit_(recommended)

## build docker image

docker build -t manga-translator:v1.0 .

## run docker image with access to gpu

docker run -v "$(pwd):/home/user" --gpus all -it manga-translator:v1.0 python3 /home/user/scripts/train.py


### Used datasets
    - Tare Himself (OG)
    - Luciano Bastos Nunes https://universe.roboflow.com/luciano-bastos-nunes/webcomics-mangs
    - Ibn Zohr University link pending
